What is CSS?

  1. CSS stands for Cascading Style Sheet

  2. CSS describe how HTML document are to be on displayed screen, paper, or in a other media

  3. CSS is not neccesary. CSS save a lot of work. It can control the layout of multiple webpages all at once.

  4. External stylesheets are stored in CSS file

CSS can be added to HTML documents in 3 ways:

  • Inline - by using the style attribute inside HTML element1
  • Internal - bu using a style element in the head section
  • External - by using a link element to link to an external CSS file
  • The most common way to add CSS, is to keep the styles in external CSS files. However, in this tutorial we will use inline and internal styles, because this is easier to demonstrate, and easier for you to try it yourself.

When comparing CSS class vs ID, the differnce is that the CSS class applies to a style to multiple element. ID, on the other hand, applies a style to one unique element. ID, is also special in that you can use a special url to link directly to an element and it's used by JavaScript.

In CSS, selectors are used to target a specific element or range of elements on a web page. Once an element has been targeted, a style or set of styles can be applied to the element.
